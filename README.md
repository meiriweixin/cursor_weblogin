# 1.项目概述：
使用Next.js 和 DeepSeek API 构建一个现代化AI聊天机器人网站

2.主要功能：
   - 用户可以在聊天界面输入消息
   - AI根据用户输入，生成实时流式回复
   - 动态显示聊天历史
   - 现代化的UI设计，包括动画效果和响应式布局

3.技术栈
Next.js, React, TypeScript, Tailwind CSS, Headless UI, Heroicons 和 Axios

4. 项目结构
使用Next.js的Pages Router，包含主页面组件，API路由，全局样式等

5. 主要功能实现：
  - 实时流式AI响应
  - 动态聊天界面
  - 响应式设计
  - 现代UI组件

6.错误处理：
实现基本的客户端和服务器端错误处理

7.未来改进计划：
包括添加用户认证，聊天历史持久化，性能优化等等
